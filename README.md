## For Developers:

Clone the source locally:

```bash
$ git clone https://github.com/arihantbansal/khalo-frontend.git
$ cd khalo-frontend
```

Install project dependencies:

```bash
$ npm install
```

Create environment variables:

1. Create file `.env`
1. Enter env variables

```
REACT_APP_DOMAIN="<!-- backend url -->"
```

Start the app:

```bash
$ npm start
```
